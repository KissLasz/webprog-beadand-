# Web-programozás 1 – Beadandó

## Projekt célja
A feladat célja egy több technológiát bemutató webalkalmazás elkészítése:
- JavaScript CRUD
- React CRUD
- SPA alkalmazás
- Fetch API és Axios használata
- OOJS alkalmazás

## Megvalósított részek

### 1. Főoldal (index.html)
- Alap struktúra elkészítve
- Menü navigáció kialakítva

### 2. JavaScript CRUD (javascript.html)
- Adatok tömbben tárolva
- CRUD műveletek működnek:
  - Create
  - Read
  - Update
  - Delete

### 3. React CRUD (react.html)
- React alapú CRUD alkalmazás
- Komponensek használata
- useState állapotkezelés

Megjegyzés:
A React alkalmazás lokálisan működik.

### 4. SPA (spa.html)
- Egyoldalas alkalmazás
- Menü alapú navigáció
- React komponensek integrálása

## Technológiák
- HTML, CSS
- JavaScript
- React (Vite alapú)
- PHP
- MySQL
(korábban volt egy bootstrap alapú változat is, de  ez elvetettük)

## Projekt struktúra
- /apps → frontend alkalmazások
- /react → React build (src + dist)
- /htdocs → tesztelt futtatási környezet

## Git használat
A projekt több lépésben lett fejlesztve:
- CRUD alapok
- React integráció
- SPA kialakítás
- Hibajavítások

A projekt jelenleg fejlesztés alatt áll, egyes részek (pl. React htdocs javítások) további finomhangolást igényelnek.
